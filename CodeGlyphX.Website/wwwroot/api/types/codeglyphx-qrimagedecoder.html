QrImageDecoder

Class Â· CodeGlyphX

Decodes QR codes from raw pixel buffers.

Examples

EXAMPLE 1


            using CodeGlyphX;
            var bytes = File.ReadAllBytes("qr.png");
            if (QrImageDecoder.TryDecodeImage(bytes, out var decoded)) {
                Console.WriteLine(decoded.Text);
            }
