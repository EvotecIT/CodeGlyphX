Properties

  • ModuleSize - Gets or sets the size of a single QR module in pixels.
  • QuietZone - Gets or sets the quiet zone size in modules.
  • Foreground - Gets or sets the foreground (dark) color.
  • Background - Gets or sets the background (light) color.
  • BackgroundGradient - Optional gradient for the background.
  • BackgroundPattern - Optional pattern overlay for the QR background area.
  • BackgroundSupersample - Background supersample factor for gradients/patterns (1 = disabled, max 4).
  • ForegroundGradient - Optional gradient for the foreground (dark) modules.
  • ForegroundPalette - Optional multi-color palette for foreground modules.
  • ForegroundPattern - Optional pattern overlay for foreground modules.
  • ForegroundPaletteZones - Optional palette overrides for specific zones.
  • Eyes - Optional eye (finder) styling overrides.
  • ModuleShape - Gets or sets the module shape.
  • ModuleScale - Gets or sets the scale of the module inside its cell (0.1..1.0).
  • ModuleScaleMap - Optional per-module scale mapping.
  • ModuleShapeMap - Optional per-module shape mapping.
  • ModuleJitter - Optional per-module jitter (organic placement).
  • ProtectFunctionalPatterns - When true, keeps non-eye functional patterns (timing/alignment/format/version/dark module) at full scale and a stable foreground color for scan reliability.
  • ProtectQuietZone - When true and a background pattern is enabled, skips drawing the pattern inside the quiet zone.
  • ModuleCornerRadiusPx - Gets or sets the corner radius in pixels for .
  • Logo - Optional logo overlay (centered).
  • Canvas - Optional canvas options for sticker-style output.
  • Debug - Optional debug overlay options.
  • PngCompressionLevel - PNG compression level (0 = stored/uncompressed, 1-9 = compressed).

Fields

  • BackgroundSupersampleMin - Minimum allowed background supersample factor.
  • BackgroundSupersampleMax - Maximum allowed background supersample factor (capped to avoid large memory spikes).
