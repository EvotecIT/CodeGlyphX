CodeGlyphX.Rendering.Ascii.MatrixAsciiRenderOptions

Rendering options for ASCII matrix output.

Methods

  • WithQuietZone - Sets the quiet zone size in modules.
    • quietZone (System.Int32)
  • WithModuleWidth - Sets the module width in characters.
    • width (System.Int32)
  • WithModuleHeight - Sets the module height in rows.
    • height (System.Int32)
  • WithScale - Sets an additional scale multiplier applied to both module width and height.
    • scale (System.Int32)
  • WithDark - Sets the character(s) used for dark modules.
    • dark (System.String)
  • WithLight - Sets the character(s) used for light modules.
    • light (System.String)
  • WithNewLine - Sets the line separator.
    • newLine (System.String)
  • WithUnicodeBlocks - Enables or disables Unicode block glyphs when defaults are used.
    • enabled (System.Boolean)
  • WithInvert - Enables or disables inverted output (dark/light swapped).
    • enabled (System.Boolean)
  • WithAnsiColors - Enables or disables ANSI color escape codes.
    • enabled (System.Boolean)
  • WithAnsiTrueColor - Enables or disables ANSI truecolor (24-bit) output.
    • enabled (System.Boolean)
  • WithAnsiDarkColor - Sets the ANSI color for dark modules.
    • color (CodeGlyphX.Rendering.Png.Rgba32)
  • WithAnsiLightColor - Sets the ANSI color for light modules.
    • color (CodeGlyphX.Rendering.Png.Rgba32)
  • WithAnsiColorizeLight - Enables or disables ANSI colorization of light modules.
    • enabled (System.Boolean)

Properties

  • QuietZone - Quiet zone size in modules.
  • ModuleWidth - Module width in characters.
  • ModuleHeight - Module height in rows.
  • Scale - Additional scale multiplier applied to both module width and height.
  • Dark - Character(s) used for dark modules.
  • Light - Character(s) used for light modules.
  • NewLine - Line separator used between rows.
  • UseUnicodeBlocks - When true, prefers Unicode block glyphs (for example, █) when defaults are used.
  • Invert - When true, swaps dark and light output.
  • UseAnsiColors - When true, emits ANSI color escape codes for dark (and optionally light) modules.
  • UseAnsiTrueColor - When true, uses 24-bit ANSI colors; otherwise maps to ANSI 256-color.
  • AnsiDarkColor - ANSI color for dark modules.
  • AnsiLightColor - ANSI color for light modules.
  • AnsiColorizeLight - When true, also colorizes light modules; otherwise leaves them uncolored.
