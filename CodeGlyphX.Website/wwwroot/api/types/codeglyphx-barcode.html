CodeGlyphX.Barcode

Simple barcode helpers with fluent and static APIs.

Use CodeGlyphX.Barcode.Save(CodeGlyphX.BarcodeType,System.String,System.String,CodeGlyphX.BarcodeOptions,CodeGlyphX.Rendering.RenderExtras) to pick the output format by file extension.
