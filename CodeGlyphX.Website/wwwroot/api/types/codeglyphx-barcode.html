Barcode

Class Â· CodeGlyphX

Simple barcode helpers with fluent and static APIs.

Use CodeGlyphX.Barcode.Save(CodeGlyphX.BarcodeType,System.String,System.String,CodeGlyphX.BarcodeOptions,System.String) to pick the output format by file extension.

Examples

EXAMPLE 1


            using CodeGlyphX;
            Barcode.Save(BarcodeType.Code128, "PRODUCT-12345", "barcode.png");
