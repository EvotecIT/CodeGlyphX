CodeGlyphX.QrEasyOptions

Simple options for QR rendering.

Properties

  • ModuleSize - Module size in pixels.
  • QuietZone - Quiet zone size in modules.
  • TargetSizePx - Target output size in pixels (0 = disabled). When set, module size is adjusted to fit this target.
  • TargetSizeIncludesQuietZone - When true, includes the quiet zone.
  • ErrorCorrectionLevel - Optional error correction level override.
  • TextEncoding - Optional text encoding override (when set, emits ECI for non-default encodings).
  • IncludeEci - When true, emits ECI headers for non-default encodings.
  • RespectPayloadDefaults - When true, payload defaults may override version/ECC settings.
  • MinVersion - Minimum QR version (1..40).
  • MaxVersion - Maximum QR version (1..40).
  • ForceMask - Optional forced mask pattern (0..7).
  • Foreground - Foreground color.
  • Background - Background color.
  • BackgroundGradient - Optional background gradient.
  • BackgroundPattern - Optional pattern overlay for the QR background area.
  • BackgroundSupersample - Background supersample factor for gradients/patterns (1 = disabled).
  • Style - Style preset for PNG rendering.
  • Art - High-level QR art options (theme + variant + intensity).
  • ArtAutoTune - When true, applies scan-safety auto-tuning for art-heavy styles.
  • ArtAutoTuneMinScore - Minimum safety score target (0..100) for art auto-tuning.
  • ModuleShape - Overrides the module shape (when set).
  • ModuleScale - Overrides the module scale (0.1..1.0).
  • ModuleScaleMap - Overrides the module scale map.
  • ModuleShapeMap - Overrides the module shape map.
  • ModuleJitter - Overrides per-module jitter options.
  • ProtectFunctionalPatterns - When true, keeps non-eye functional patterns at a stable, scan-friendly style.
  • ProtectQuietZone - When true and a background pattern is enabled, preserves a clean quiet zone.
  • ModuleCornerRadiusPx - Overrides the module corner radius in pixels.
  • ForegroundGradient - Overrides the foreground gradient.
  • ForegroundPalette - Overrides the foreground palette.
  • ForegroundPattern - Overrides the foreground pattern overlay.
  • ForegroundPaletteZones - Overrides palette zones.
  • Canvas - Optional canvas options for sticker-style output.
  • Debug - Optional debug overlay options (PNG only).
  • Eyes - Overrides eye (finder) styling.
  • LogoPng - Optional logo PNG bytes (embedded for PNG/SVG/HTML).
  • LogoScale - Logo size relative to the QR area (excluding quiet zone).
  • LogoPaddingPx - Padding around the logo in pixels.
  • LogoDrawBackground - Whether to draw a background plate behind the logo. When enabled, the encoder may auto-bump the minimum version for scan safety.
  • AutoBumpVersionForLogoBackground - When true, bumps to a safer minimum for logo background plates.
  • LogoBackgroundMinVersion - Minimum version to use when a logo background plate is enabled (0 = disable auto bump).
  • LogoBackground - Logo background color (defaults to QR background).
  • LogoCornerRadiusPx - Logo background corner radius in pixels.
  • JpegQuality - JPEG quality (1..100).
  • WebpQuality - WebP quality (0..100). A value of 100 uses lossless VP8L.
  • IcoSizes - ICO output sizes in pixels (1..256). Defaults to common icon sizes.
  • IcoPreserveAspectRatio - When true, preserves aspect ratio and pads to square for ICO.
  • HtmlEmailSafeTable - When true, renders HTML using email-safe tables.
