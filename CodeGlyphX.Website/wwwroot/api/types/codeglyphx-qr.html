QR

Class Â· CodeGlyphX

Simple QR helpers with fluent and static APIs.

Use CodeGlyphX.QR.Save(System.String,System.String,CodeGlyphX.QrEasyOptions,System.String) to pick the output format by file extension.

Examples

EXAMPLE 1


            using CodeGlyphX;
            QR.Save("https://example.com", "qr.png");
