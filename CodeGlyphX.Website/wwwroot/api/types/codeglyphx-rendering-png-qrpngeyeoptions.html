CodeGlyphX.Rendering.Png.QrPngEyeOptions

Styling overrides for QR finder (eye) patterns.

Properties

  • UseFrame - Gets or sets whether to draw each eye as a single frame (outer ring + inner dot).
  • FrameStyle - Gets or sets the eye frame style when is enabled.
  • OuterShape - Gets or sets the outer (7x7) eye module shape.
  • InnerShape - Gets or sets the inner (3x3) eye module shape.
  • OuterScale - Gets or sets the scale of the outer modules inside their cells (0.1..1.0).
  • InnerScale - Gets or sets the scale of the inner modules inside their cells (0.1..1.0).
  • OuterCornerRadiusPx - Gets or sets the corner radius for outer modules (pixels).
  • InnerCornerRadiusPx - Gets or sets the corner radius for inner modules (pixels).
  • OuterColor - Optional outer eye color override.
  • OuterColors - Optional per-eye outer colors (TopLeft, TopRight, BottomLeft).
  • InnerColor - Optional inner eye color override.
  • InnerColors - Optional per-eye inner colors (TopLeft, TopRight, BottomLeft).
  • OuterGradient - Optional gradient for the outer frame.
  • OuterGradients - Optional per-eye outer gradients (TopLeft, TopRight, BottomLeft).
  • InnerGradient - Optional gradient for the inner dot.
  • InnerGradients - Optional per-eye inner gradients (TopLeft, TopRight, BottomLeft).
  • GlowRadiusPx - Optional glow radius in pixels when is . A value of 0 uses a reasonable default based on module size.
  • GlowColor - Optional glow color override. Defaults to the outer eye color.
  • GlowAlpha - Maximum glow alpha (0..255) when is .
  • SparkleCount - Optional sparkle count drawn around the eyes on the canvas.
  • SparkleRadiusPx - Sparkle radius in pixels.
  • SparkleSpreadPx - How far sparkles can extend beyond the eye ring (in pixels).
  • SparkleColor - Sparkle color override. Defaults to the outer eye color.
  • SparkleSeed - Random seed for sparkle placement. Use 0 to auto-randomize per render.
  • SparkleProtectQrArea - When true (default), sparkles do not draw inside the QR area.
  • SparkleAllowOnQrBackground - When false (default), sparkles render only when a canvas is enabled.
  • AccentRingCount - Optional accent ring count drawn around the eyes on the canvas.
  • AccentRingThicknessPx - Accent ring stroke thickness in pixels.
  • AccentRingSpreadPx - How far accent rings can extend beyond the eye ring (in pixels).
  • AccentRingJitterPx - Small jitter applied to ring radius (in pixels).
  • AccentRingColor - Accent ring color override. Defaults to the outer eye color with a softer alpha.
  • AccentRingSeed - Random seed for accent ring placement. Use 0 to auto-randomize per render.
  • AccentRingProtectQrArea - When true (default), accent rings do not draw inside the QR area.
  • AccentRingAllowOnQrBackground - When false (default), accent rings render only when a canvas is enabled.
  • AccentRayCount - Optional accent ray count drawn around the eyes on the canvas.
  • AccentRayLengthPx - Accent ray length in pixels.
  • AccentRayThicknessPx - Accent ray stroke thickness in pixels.
  • AccentRaySpreadPx - How far accent rays can start beyond the eye ring (in pixels).
  • AccentRayJitterPx - Small jitter applied to ray start radius (in pixels).
  • AccentRayLengthJitterPx - Small jitter applied to ray length (in pixels).
  • AccentRayColor - Accent ray color override. Defaults to the outer eye color with a softer alpha.
  • AccentRaySeed - Random seed for accent ray placement. Use 0 to auto-randomize per render.
  • AccentRayProtectQrArea - When true (default), accent rays do not draw inside the QR area.
  • AccentRayAllowOnQrBackground - When false (default), accent rays render only when a canvas is enabled.
  • AccentStripeCount - Optional accent stripe count drawn around the eyes on the canvas. Stripes are short tangential strokes that hug the eye perimeter.
  • AccentStripeLengthPx - Accent stripe length in pixels.
  • AccentStripeThicknessPx - Accent stripe stroke thickness in pixels.
  • AccentStripeSpreadPx - How far accent stripes can extend beyond the eye ring (in pixels).
  • AccentStripeJitterPx - Small jitter applied to stripe radius (in pixels).
  • AccentStripeLengthJitterPx - Small jitter applied to stripe length (in pixels).
  • AccentStripeColor - Accent stripe color override. Defaults to the outer eye color with a softer alpha.
  • AccentStripeSeed - Random seed for accent stripe placement. Use 0 to auto-randomize per render.
  • AccentStripeProtectQrArea - When true (default), accent stripes do not draw inside the QR area.
  • AccentStripeAllowOnQrBackground - When false (default), accent stripes render only when a canvas is enabled.
