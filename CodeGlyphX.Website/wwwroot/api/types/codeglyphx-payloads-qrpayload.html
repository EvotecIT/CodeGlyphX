QrPayload

Class · CodeGlyphX.Payloads

Helpers for building common QR payload strings (URLs, Wi‑Fi, vCard, and basic URI schemes).

The returned strings can be passed directly to CodeGlyphX.QrCodeEncoder.EncodeText(System.String,CodeGlyphX.QrErrorCorrectionLevel,System.Int32,System.Int32,System.Nullable{System.Int32}).
