CodeGlyphX.MicroQrDecoded

Result of decoding a Micro QR code payload.

Properties

  • Version - Gets the decoded Micro QR version (1..4).
  • ErrorCorrectionLevel - Gets the decoded error correction level.
  • Mask - Gets the decoded mask pattern (0..3).
  • Bytes - Gets the decoded payload bytes.
  • Text - Gets the decoded payload interpreted as text.
