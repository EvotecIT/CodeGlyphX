BarcodeEasy

Class Â· CodeGlyphX

One-line barcode helpers with sane defaults.

Examples

EXAMPLE 1


            using CodeGlyphX;
            var png = BarcodeEasy.RenderPng(BarcodeType.Code128, "PRODUCT-12345");
            var svg = BarcodeEasy.RenderSvg(BarcodeType.Code128, "PRODUCT-12345");
